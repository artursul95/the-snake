# Игра змейка
Данная игра написана на языке программирования Python с использованием принципов ООП.
# Как играть
Клонировать репозиторий к себе локально и установить зависимости из файла requirement.txt с помощью команд:
git clone https://github.com/artursul95/the-snake
pip install -r requirements.txt
Далее просто необходимо запустить исполняемый файл the_snake.py
